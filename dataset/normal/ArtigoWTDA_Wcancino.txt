O reconhecimento de padrões biológicos é uma tarefa difícil pois, durante a evolução, estes padrões são modificados e alongados, deixando de apresentar-se em sua forma original.  Uma forma de tratar este problema é mediante o alinhamento de seqüencias, que consiste em alinhar um padrão biológico sobre a seqüência objetivo, procurando semelhanças entre o padrão e a seqüência.
Para obter os melhores alinhamentos é necessário uma pontuação para cada solução possível. Assim, o problema de alinhamento de seqüências pode ser tratado como um problema de otimização, com a procura de soluções de melhor pontuação. A aplicação de técnicas de Computação Evolutiva surge como uma alternativa para resolver esse problema.
