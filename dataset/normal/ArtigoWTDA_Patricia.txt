Muitos esforços vêm sendo aplicados para encontrar novas metodologias para a visualização de dados volumétricos. A visualização volumétrica apoia múltiplas áreas do conhecimento, desde a medicina até as simulações de fluidos, sendo uma das principais técnicas de visualização o Rendering.
Dentro do Rendering destacam-se dois métodos. O Rendering de Superficies (RS), que obtem imagens das superfícies limitantes do conjunto de dados de forma rápida e efetiva, sendo uma das desvantagens a perda de informação volumétrica interna. Já, no caso do Rendering Volumétrico Direto (RVD), cria-se uma imagem de ótima qualidade diretamente a partir do volume de dados tridimensional, mas com um alto custo computacional o que dificulta a interação com o volume.
